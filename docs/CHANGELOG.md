# Changelog

## [v2.5.0] - 2026-03-15

### Added
- **群聊置顶**: 为群聊增加了置顶功能。
- **联系人多选**: 为联系人页面增加了多选功能。
- **表情包系统**: 
  - 增加了表情包上传、浏览、UI 面板选择功能。
  - 支持固定描述列表按名称包含匹配。

### Changed
- **写时随机机制**: 修复了历史表情每次刷新变脸的问题。拦截 LLM 文本，将表情描述（如 `[表情]开心`）随机替换为绝对路径后入库，读取时保持确定性。
- **记忆优化**: 修改了群聊中发送消息前的记忆总结逻辑。群聊前会总结：
  - 群成员的单聊近况。
  - 群成员参与的**其他群聊**的动态。
  - **跳过**当前群聊（避免记忆回声）。

## [v2.4.1] - 2026-03-14

### Changed
- 完善朋友圈点赞、评论动效。
- 修复部分内存与数据库查询性能。

## [v2.4.0] - 2026-03-04

### Added
- **朋友圈交互与记忆总结增强**:
  - 用户可以点赞和评论朋友圈。
  - AI 可主动发表评论并根据人设回复用户的评论。
  - 互动后自动触发 AI 提炼一句话总结，写入当日的短期记忆 `6_memory_short.json`。

## [v2.3.0] - 2026-03-02

### Added
- **拍一拍系统**: 双击头像发送拍一拍，支持群聊与单聊，支持人设与多语言后缀配置。
- **记忆连贯同步**:
  - 单聊前自动同步群聊记忆。
  - 群聊前自动同步各成员的单聊记忆。

### Changed
- **年龄管理**: 从 persona 中剥离，依据配置中的生日每年 1/1 自动增加 1 岁。
- **气泡布局优化**: 头像内侧横向并排显示多条连续的气泡消息。
- 空消息不再展示，优化了部分群聊拍一拍的响应规则。

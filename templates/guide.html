GETTING STARTED

从注册到开始使用

按照下面的步骤完成账号、个人信息、消息通知、线路与模型配置。

⌕
登录 / 注册 配置个人信息 接收通知 线路与模型 错误码与熔断 安装成应用

1. 登录 / 注册

注册账号

打开注册页面，填写邮箱、用户名和密码后提交。

邮箱形式不受限制，格式不正确也可以注册。但如果之后忘记密码，将无法通过邮箱找回。遇到这种情况，请联系管理员迁移账号。

登录账号

打开登录页面，填写注册时使用的邮箱和密码即可登录。

2. 配置个人信息

  1. 个人主页：设置头像和 name（显示名称）。
  2. 个人主页 → 账号与通知设置：设置年龄、拍一拍后缀和语言。
  3. 个人主页 → 全局用户设定 → 编辑 Edit：填入你自己的人设，完成后保存设定。

3. 接收通知

  1. 邮箱通知：进入个人主页 → 账号与通知设置，在“通知邮箱”中填入用于接收通知的邮箱。
  2. 浏览器消息推送：进入个人主页 → 浏览器消息推送，点击“开启通知权限”。推荐使用 Firefox（火狐）浏览器，其他浏览器可能开启失败。
同一设备使用多个账号：请分别登录每个需要接收推送的账号，并在每个账号中单独点击一次“开启通知权限”。同一设备可以同时接收这些已主动开启账号的通知。

5. 错误码、报错与熔断机制

遇到 API 报错时，先停止连续重试，记下错误码并确认自己正在使用 Gemini 直连还是国内中转。不同线路的同一个错误码可能代表不同问题。

请勿把完整 API Key、登录密码或含有 Key 的截图发给他人。联系管理员时只需提供错误码、所用线路、发生时间和已遮盖敏感信息的截图。

熔断是怎么工作的？

熔断用于阻止错误配置持续请求服务商。它按“当前用户 + API 线路 + 同一错误码”分别计数，只影响出错的线路，不会冻结整个账号。

第 1 次：警告 立即停止重试，根据错误码检查 Key、余额、模型 ID 或线路选择。
第 2 次：冷却 10 分钟 倒计时结束前不要反复发送。利用冷却时间修正配置，再尝试一次。
第 3 次：禁用该 API 线路 先修正配置，再联系管理员解除。另一条配置正确的线路仍可使用。
非禁用状态下，某线路成功调用后会清除该线路的临时计数；同一错误超过约 30 分钟未再次触发，也会自动复位。已被禁用的线路不会自动恢复，需要管理员解除。
国内中转常见错误码
401身份验证失败 计入熔断

检查个人主页 → 账号与通知设置 → OpenRouter是否填入了当前中转商的 Key。重新复制时不要带空格或换行。

402账户点数或余额不足 计入熔断

登录正在使用的 API 服务商检查余额。完成充值后先确认到账，再返回重试。

403模型代码或 Key 权限异常 计入熔断

确认中转商选择正确、模型 ID 完整且该 Key 有权使用此模型。优先从文档的可用模型列表复制。

525中转服务器连接异常 计入熔断

通常是中转服务器连接问题。稍后再试；持续出现时记录时间并联系管理员。

429请求过于频繁 不计入熔断

停止发送并等待至少一分钟，之后放慢发送频率。

524AI 思考或请求超时 不计入熔断

缩短本次聊天内容，精简角色或全局人设后再试。

500+服务商繁忙或临时故障 通常不计入熔断

等待几分钟再试，不要连续点击发送。若长时间持续，联系管理员并提供具体错误码。

Cloudflare 安全拦截

系统会暂停所有国内中转请求约 3 分钟。这是服务器级临时保护，不一定是你的 Key 错误；等待倒计时结束即可。

Gemini 直连常见错误码
400请求参数异常 计入熔断

先确认使用文档中的推荐模型配置。仍然出现时不要反复发送，联系管理员检查请求配置。

403访问被拒绝 计入熔断

检查个人主页 → 账号与通知设置 → Gemini Key是否正确，重新复制时不要带空格或换行。

429访问频率受限 不计入熔断

停止发送并等待一段时间，之后降低发送频率。

500 / 503 / 504谷歌服务临时故障 不计入熔断

系统会先自动重试；仍失败时等待几分钟。持续超时可精简聊天内容和人设。

其他常见提示
连接超时 / 网络连接波动

检查当前网络，稍后重新发送一次。不要连续点击发送，以免造成重复消息或触发服务商限流。

AI 返回无法解析的异常信号

通常是上游返回内容异常。稍后重试；反复出现时记录所用模型并联系管理员。

登录失效 / Unauthorized / 401

如果是在页面接口而不是 AI 中转提示中看到，请重新登录账号，然后回到原页面重试。

记忆同步失败

本次回复仍可能生成，但可能缺少部分单聊或群聊上下文。可稍后重试；持续出现时联系管理员。

线路被禁用后怎么办？

  1. 不要继续重复发送，先根据最后一个错误码修正 Key、余额、模型或线路设置。
  2. 可以切换到另一条已经正确配置的 API 线路继续使用；线路禁用不等于账号冻结。
  3. 确认配置已经修正后联系管理员解除，并提供错误码、线路、发生时间和已遮盖敏感信息的截图。
  4. 解除后先发送一条短消息测试。若仍报同一错误，停止重试并再次检查配置。

6. 把网站安装成 PWA 应用

把“Sakura 樱语”添加到手机桌面后，可以像打开普通 App 一样点击图标进入，不必每次重新寻找网址。这是网站的 PWA 版本，不需要下载 APK，也不需要经过应用商店。

PWA 仍然需要网络才能正常聊天。安装只会添加应用入口，不会创建新账号；请继续使用原来的邮箱和密码登录。
1 打开网站 在手机浏览器访问 Sakura 樱语首页，等待页面完整加载。
2 打开浏览器菜单 Android 点击“⋮”或菜单按钮；iPhone Safari 点击分享按钮。
3 确认安装 选择“安装应用”或“添加到主屏幕”，确认后桌面会出现“樱语”图标。

Android · Chrome

  1. 使用 Chrome 打开 Sakura 樱语首页。
  2. 点击地址栏右侧或右上角的三个点“⋮”。
  3. 选择添加到主屏幕 → 安装；部分版本会直接显示“安装应用”。
  4. 按照屏幕提示确认，返回桌面查看“樱语”图标。

Android · Firefox（推荐）

  1. 使用 Firefox 打开 Sakura 樱语首页。
  2. 点击浏览器的菜单按钮。
  3. 选择安装。如果只看到“添加到主屏幕”，选择它也可以。
  4. 点击“自动添加”，或按照提示把图标拖到桌面。

iPhone / iPad · Safari

  1. 请使用 Safari 打开 Sakura 樱语首页。
  2. 点击分享按钮（方框上方有一个向上的箭头）；部分 Safari 布局需要先点“更多”，再点“分享”。
  3. 向下滚动，选择添加到主屏幕。
  4. 如果看到“作为 Web App 打开”，将它开启，然后点击右上角“添加”。

其他 Android 浏览器

  1. 打开网站后点击右上角“三个点”或“三条横线”。
  2. 寻找“安装应用”“添加到桌面”“添加到主屏幕”等选项。
  3. 确认名称和图标后完成添加。不同手机品牌的文字和位置可能略有不同。
找不到安装入口怎么办？
菜单里没有“安装应用”

继续寻找“添加到主屏幕”或“添加至桌面”。先等待网站完整加载并刷新一次，也可以更新浏览器后重试。

iPhone 没有“添加到主屏幕”

确认正在使用 Safari。在分享菜单底部选择“编辑操作”，把“添加到主屏幕”加入可用操作。

安装后仍显示浏览器地址栏

当前添加的可能只是网页快捷方式。删除该图标后重新操作，并优先选择“安装应用”或开启“作为 Web App 打开”。

安装后无法离线聊天

这是正常现象。PWA 提供接近 App 的打开方式，但聊天、同步和 AI 回复仍需要网络。

iPhone / iPad 如需浏览器消息推送，请先把网站添加到主屏幕，再从桌面的“樱语”图标打开，然后进入个人主页 → 浏览器消息推送开启权限。
